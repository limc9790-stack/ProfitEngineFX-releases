## v1.6.3 — Daily Reset Window settings

- Added **Daily Reset Window** settings (Settings → Risk Management). Configurable start/end hour for the daily-loss counter window. Default: **00:00 → 24:00 SGT** (full calendar day). Examples in the UI: `22:00 → 22:00` for broker forex day (NY close, wraps midnight); `09:00 → 17:00` for work-hours only.
- Daily-loss counter now uses **SGT calendar day** by default (was previously the broker-day window starting 22:00 SGT).
